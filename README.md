## Install Crypto-Creeper via Terminal.app
```
git clone https://github.com/se7enack/Crypto-Creeper.git
cd Crypto-Creeper/mac-binary
unzip cryptocreeper-v1.0.1.zip
mv Crypto\ Creeper.app ~/Applications/.
```
